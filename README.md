## 🚀 Menjalankan di Lokal

Karena project ini murni HTML/CSS/JS tanpa build tool, gak perlu instalasi apapun:

```bash
git clone https://github.com/rmzairin/Portfolio-Web-Skirk.git
cd Portfolio-Web-Hu-Tao
```

Lalu tinggal buka `index.html` langsung di browser, atau kalau mau lewat local server:

```bash
# pakai Python
python -m http.server 5500

# atau pakai ekstensi Live Server di VS Code
```
